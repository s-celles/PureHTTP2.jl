# Contributors

PureHTTP2.jl exists thanks to the people listed here. Contributions
include code, tests, documentation, bug reports, reviews, and
upstream coordination.

## Maintainers

- **Sébastien Celles** ([@s-celles](https://github.com/s-celles)) —
  project lead, original author, package extraction from
  [gRPCServer.jl](https://github.com/s-celles/gRPCServer.jl).

## Upstream heritage

PureHTTP2.jl was lifted and shifted from the `http2` submodule of
[gRPCServer.jl](https://github.com/s-celles/gRPCServer.jl) at commit
[`4abc0932`](https://github.com/s-celles/gRPCServer.jl/tree/4abc09324736b3597da5502385dbce24a1edb174).
The license, copyright, and development history of that submodule
carry over to this package. See the Provenance appendix in
[`CHANGELOG.md`](CHANGELOG.md) for details.

## How to be listed

Anyone who has a merged pull request, a filed issue that led to a
fix, or a reviewed change is welcome to add themselves to this file
in the section below. Open a PR updating this file as part of (or
following) your contribution.
